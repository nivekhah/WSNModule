
## WSNModule

#### 🌏 无线传感网可视化模型

一个基于Java的无线传感网可视化模型。

#### 🔨 主要功能

- 能量动态采集和消耗
- 节点位置连续移动
- 节点间消息传递
- 秒级可视化

#### 📕 目录结构

- Config，系统参数配置
- Unity，系统实体类
- Utils，各实体工具类
  - AlgoUtil，在此修改镶嵌节点算法
